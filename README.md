# tmf.RealEstateSales

This project analyzes trends from real estate sales data spanning multiple years. The goal is to explore the long-term trends in property values, sales ratios and time on the market.  

This analysis will be perfomed using Python and presented using Power BI.

Data was sourced from Kaggle : https://www.kaggle.com/datasets/omniamahmoudsaeed/real-estate-sales-2001-2022?resource=download
